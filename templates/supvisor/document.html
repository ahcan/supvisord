HƯỚNG DẪN SỬ DỤNG


Lưu ý chung khi sử dụng tool: không được gõ tiếng việt có dấu, kí tự đặc biệt, kí tự khoảng trắng. Tên hiển thị sẽ được tự động thêm vào ".ini" sau khi được khởi tạo.

  • TẠO PROCESS MỚI

    • Đăng nhập --> Chọn "Add new process" --> nhập thông tin (stream type, name, syteam key, ip) --> chọn nút "Save anh Start". chờ web load 3-5s process sẽ được tạo mới và hoạt động.

  • START, RESTART PROCESS SẴN CÓ

    • Đăng nhập --> Chọn "Add new process" --> nhập thông tin (stream type, syteam key, ip), nếu không thay đổi thông tin thì process sẽ được start với thông tin cũ --> chọn nút "Save anh Start". chờ web load 3-5s process sẽ được tạo mới và hoạt động.

  • GIAO DIỆN CHÍNH

    • SAU 10S TRANG WEB SẼ TỰ ĐỘNG CẬP NHẬT THÔNG TIN MỚI
    State: Trạng thái process (running, stop, ..)
    Description: Mô tả chi tiết trạng thái của process, thời gian chạy (Uptime), nếu thời gian chạy của process đạt ngưỡng 30s rồi tự về 0s --> process dang có lỗi.
    Command: Hiển thị nội dung câu lệnh của process, đồng thời cho biết ip và stream key đang dùng cho process này.
    Action: điều khiển start, stop, delete process.
    Add new process: thêm mới process.

  • GIAO DIỆN THÊM MỚI PROCESS

    • Stream type: chọn loại streaming (Facebook, Youtube) để khởi tạo câu lệnh tương ứng.
    Name: Tên hiển thị của process
    Stream key (khóa luồng): Mỗi lần stream Facebook, youtube sẽ cung cấp thông tin stream key khác nhau, tốt nhất mỗi lần start process nên nhập mới stream key và kiểm tra thông tin chính xác.
    IP:Port: Nhập ip luồng cần stream vd: 224.1.1.1:1234. nếu thông tin này bỏ trống hệ thống sẽ tạo process mới với stream key mặt định là 225.1.1.7:30120.
    Nút "Save anh Start": lưu lại thông tin đồng thời start process, trang web sẽ load chậm từ 3-5s (thời gian chờ các tiến tình cập nhật và khởi tạo luồng mới xảy ra bên trong server).

  • GIAO DIỆN START, RESTART PROCESS CÓ SẴN

    • Stream type: hệ thống sẽ tự chọn stream type tương ứng, nếu stream type là "UNKNOW" bạn cần chọn đáng stream type bạn cần.
    Stream key (khóa luồng): Mỗi lần stream Facebook, youtube sẽ cung cấp thông tin stream key khác nhau, tốt nhất mỗi lần start process nên nhập mới stream key và kiểm tra thông tin chính xác.
    IP:Port: hệ thống sẽ hiện thì thông tin ip cũ, nhập ip hợp lệ vd: 224.1.1.1:1234
    Nút "Save anh Start": lưu lại thông tin đồng thời start process, trang web sẽ load chậm từ 3-5s (thời gian chờ các tiến tình cập nhật và khởi tạo luồng mới xảy ra bên trong server).
